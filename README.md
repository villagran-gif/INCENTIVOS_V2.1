# Zendesk Sell Private App - INCENTIVOS Resumen Mensual (Top Bar)

Este ZIP está listo para subir como *Private App* en Zendesk Sell.

Requisitos del ZIP:
- `manifest.json` debe estar en la **raíz** del ZIP.
- La carpeta `assets/` también en la raíz.

La app consulta: /api/monthly/YYYY-MM en tu servicio Render y muestra el resumen.
